# Story 1.4 — Layout Base da Academy + Identidade Visual
**Epic:** EPIC-1 Foundation  
**Status:** Ready for Review  
**Complexity:** S (3 pts)  
**Priority:** P0

## Descrição
Como médico cliente, quero que a plataforma transmita a identidade profissional da Carmem Cavalcante desde o primeiro acesso, com layout mobile-first, navegação clara e visual premium.

## Acceptance Criteria

**Given** o médico está autenticado  
**When** acessar qualquer página da Academy  
**Then:**
- [x] Header com logo Carmem Cavalcante Academy + nome do usuário + botão logout
- [x] Navegação mobile: bottom navigation com: Home · Perfil
- [x] Navegação desktop: sidebar ou top nav
- [x] Cores: azul primário `#01254C` oklch(0.301 0.079 248) via CSS var --primary
- [x] Fonte profissional (Inter — Geist como fallback)
- [x] Layout responsivo funcionando em 375px (iPhone SE)
- [x] Loading state com skeleton (não spinner) para carregamento de dados

## Scope
**IN:**
- `app/(academy)/layout.tsx` com header + nav
- Componentes: Header, BottomNav (mobile), Sidebar (desktop)
- Tokens de cor Tailwind configurados
- Fonte web carregada via `next/font`
- Skeleton loader genérico reutilizável

**OUT:**
- Conteúdo das páginas (outras stories)
- Layout admin (Story 1.5)

## Dependencies
- Story 1.1 (Shadcn/UI + Tailwind configurados)
- Story 1.3 (auth + logout funcionando)

## Definition of Done
- [x] Layout renderiza em 375px sem scroll horizontal
- [x] Logo aparece no header
- [x] Logout funciona
- [x] `npm run lint` passa
