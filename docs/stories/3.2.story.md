# Story 3.2 — Página de Trilha (Módulos + Lista de Aulas)
**Epic:** EPIC-3 Core Learning  
**Status:** Ready  
**Complexity:** M (3 pts)  
**Priority:** P0

## Descrição
Como médico cliente, quero ver todos os módulos e aulas de uma trilha com o status de cada aula (não iniciada, em andamento, concluída) para planejar meu aprendizado.

## Acceptance Criteria

**Given** médico clica em uma trilha  
**When** acessar `/trilha/[id]`  
**Then:**
- [ ] Título e descrição da trilha
- [ ] Barra de progresso geral da trilha (% concluído)
- [ ] Módulos listados em ordem com suas aulas
- [ ] Cada aula com: ícone de status (não iniciada/em andamento/concluída), título, duração, lock indicator se `is_locked=true`
- [ ] Aula bloqueada pelo NPS gate exibe ícone de cadeado
- [ ] Clicar em aula navega para `/aula/[id]`
- [ ] Botão "Continuar de onde parou" destacado para a última aula em andamento

## Scope
**IN:**
- Listagem de módulos + aulas com status
- Lock indicator para aulas com `lock_condition='nps'`
- Navegação para aula

**OUT:**
- Player de vídeo (Story 3.3)

## Dependencies
- Story 1.2, Story 3.1

## Definition of Done
- [ ] Status das aulas carregado do banco
- [ ] Lock indicator funcionando
- [ ] `npm run typecheck` passa
