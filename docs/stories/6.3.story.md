# Story 6.3 — DNS + Domínio + Deploy Produção Vercel
**Epic:** EPIC-6 Launch  
**Status:** Ready  
**Complexity:** S (2 pts)  
**Priority:** P0

## Descrição
Como equipe, precisamos colocar a plataforma no ar em carmemcavalcanteacademy.com.br com SSL ativo e variáveis de ambiente de produção configuradas.

## Acceptance Criteria

**Given** o código está no repositório GitHub main branch  
**When** o deploy de produção for configurado  
**Then:**
- [ ] Domínio `carmemcavalcanteacademy.com.br` configurado no Vercel
- [ ] DNS configurado (registrar): A record ou CNAME apontando para Vercel
- [ ] SSL/HTTPS ativo e certificado válido
- [ ] Variáveis de ambiente de produção no Vercel:
  - `NEXT_PUBLIC_SUPABASE_URL`
  - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
  - `SUPABASE_SERVICE_ROLE_KEY`
  - `RESEND_API_KEY`
- [ ] CI/CD automático: push para `main` → deploy automático no Vercel
- [ ] Supabase de produção configurado (não o de desenvolvimento)
- [ ] Domínio de e-mail Resend configurado (SPF/DKIM)

## Scope
**IN:**
- Configuração de domínio no Vercel
- Variáveis de ambiente de produção
- Configuração de DNS
- Verificação de e-mail Resend em produção

**OUT:**
- CDN personalizada
- WAF / DDoS protection

## Dependencies
- Story 6.1 (QA), Story 6.2 (conteúdo seed)
- Domínio registrado (responsabilidade da equipe Carmem)

## Definition of Done
- [ ] https://carmemcavalcanteacademy.com.br carregando em produção
- [ ] SSL verde no browser
- [ ] Magic link funciona em produção
- [ ] E-mail de boas-vindas chega na caixa de entrada (não spam)
