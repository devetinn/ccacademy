# Story 1.1 — Setup do Projeto
**Epic:** EPIC-1 Foundation  
**Status:** Ready for Review  
**Complexity:** M (3 pts)  
**Priority:** P0

## Descrição
Como desenvolvedor, preciso criar o projeto Next.js 15 com toda a stack configurada e pronta para desenvolvimento, incluindo integração com Supabase, Shadcn/UI, Tailwind e estrutura de pastas conforme o PRD.

## Acceptance Criteria

**Given** o repositório está vazio  
**When** o setup estiver completo  
**Then:**
- [x] `npx create-next-app` com App Router + TypeScript executado
- [x] Supabase JS client configurado (`@supabase/ssr`, `@supabase/supabase-js`)
- [x] Shadcn/UI inicializado com tema customizado (azul Carmem como primary)
- [x] Tailwind CSS configurado com tokens de design (cores, fontes)
- [x] Variáveis de ambiente: `.env.local` com `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `RESEND_API_KEY`
- [x] `react-player`, `react-pdf`, `framer-motion` instalados
- [x] Estrutura de pastas criada conforme PRD seção 7
- [x] `lib/supabase/client.ts` — `createBrowserClient`
- [x] `lib/supabase/server.ts` — `createServerClient`
- [x] `npm run dev` inicia sem erros
- [x] `npm run lint` e `npm run typecheck` passam

## Dev Notes
- Usar `@supabase/ssr` (não o pacote legado `auth-helpers`)
- Tokens de cor do Tailwind: `--primary: oklch(0.301 0.079 248)` = #01254C (confirmado)
- Mobile-first: `--breakpoint-xs: 23.4375rem` (375px) adicionado via `@theme` em globals.css
- Shadcn v4 usa Tailwind v4 — não existe tailwind.config.ts, tudo via CSS custom properties
- Versão efetiva do Next.js: 16.2.6 (latest ao momento do setup)
