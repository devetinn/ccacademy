# Story 1.1 — Setup do Projeto
**Epic:** EPIC-1 Foundation  
**Status:** Ready  
**Complexity:** M (3 pts)  
**Priority:** P0

## Descrição
Como desenvolvedor, preciso criar o projeto Next.js 15 com toda a stack configurada e pronta para desenvolvimento, incluindo integração com Supabase, Shadcn/UI, Tailwind e estrutura de pastas conforme o PRD.

## Acceptance Criteria

**Given** o repositório está vazio  
**When** o setup estiver completo  
**Then:**
- [ ] `npx create-next-app` com App Router + TypeScript executado
- [ ] Supabase JS client configurado (`@supabase/ssr`, `@supabase/supabase-js`)
- [ ] Shadcn/UI inicializado com tema customizado (azul Carmem como primary)
- [ ] Tailwind CSS configurado com tokens de design (cores, fontes)
- [ ] Variáveis de ambiente: `.env.local` com `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `RESEND_API_KEY`
- [ ] `react-player`, `react-pdf`, `framer-motion` instalados
- [ ] Estrutura de pastas criada conforme PRD seção 7
- [ ] `lib/supabase/client.ts` — `createBrowserClient`
- [ ] `lib/supabase/server.ts` — `createServerClient`
- [ ] `npm run dev` inicia sem erros
- [ ] `npm run lint` e `npm run typecheck` passam

## Scope
**IN:**
- Next.js 15 App Router + TypeScript
- Supabase SSR package + clients
- Shadcn/UI + Tailwind com tema Carmem
- Instalação de todas as dependências do PRD
- Estrutura de pastas base

**OUT:**
- Qualquer feature de produto
- Migrations do banco (Story 1.2)
- Autenticação (Story 1.3)

## Dependencies
- Conta Supabase criada com projeto ativo
- Chaves de API Supabase disponíveis
