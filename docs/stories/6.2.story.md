# Story 6.2 — Seed das Primeiras Trilhas (3 trilhas V1)
**Epic:** EPIC-6 Launch  
**Status:** Ready  
**Complexity:** S (2 pts)  
**Priority:** P0

## Descrição
Como equipe Carmem, precisamos popular a plataforma com as 3 trilhas iniciais do V1 com conteúdo real antes do lançamento.

## Acceptance Criteria

**Given** a plataforma está em produção  
**When** o seed for executado  
**Then:**
- [ ] Trilha 1: "Primeiros Passos no App" — criada com módulos e aulas reais
- [ ] Trilha 2: "Entendendo Seu Faturamento" — criada com módulos e aulas reais
- [ ] Trilha 3: "Glosas e Recursos" — criada com módulos e aulas reais
- [ ] Vídeos do YouTube (não-listados) linkados em cada aula
- [ ] Materiais de apoio (PDFs) carregados para pelo menos a primeira aula de cada trilha
- [ ] 1 admin criado (e-mail da equipe Carmem)
- [ ] 1 médico de teste convidado (para validar o fluxo end-to-end)

## Scope
**IN:**
- Inserção de dados via admin CMS (interface visual)
- Ou script SQL de seed se mais eficiente

**OUT:**
- Conteúdo dos vídeos (responsabilidade da equipe Carmem)

## Dependencies
- EPIC-2 (CMS Admin funcional), Story 6.1 (QA concluído)
- Vídeos gravados e carregados no YouTube pela equipe Carmem

## Definition of Done
- [ ] 3 trilhas visíveis na home para o médico de teste
- [ ] Fluxo end-to-end validado com médico de teste real
