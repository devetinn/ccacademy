# Story 1.5 — Layout Base Admin + Navegação
**Epic:** EPIC-1 Foundation  
**Status:** Ready  
**Complexity:** S (2 pts)  
**Priority:** P1

## Descrição
Como membro da equipe Carmem, quero um backoffice com navegação clara entre as seções de gerenciamento (trilhas, usuários, Q&A, NPS, métricas).

## Acceptance Criteria

**Given** usuário com role=admin está autenticado  
**When** acessar `/admin/*`  
**Then:**
- [ ] Sidebar com itens: Dashboard · Trilhas · Usuários · Perguntas · NPS
- [ ] Indicador visual da seção ativa
- [ ] Header com "Admin — Carmem Cavalcante Academy" + nome do usuário
- [ ] Layout funciona no desktop (min 1024px é o uso principal do admin)
- [ ] Breadcrumb em páginas aninhadas

## Scope
**IN:**
- `app/(admin)/layout.tsx` com sidebar
- Componente AdminSidebar com links e active state

**OUT:**
- Conteúdo das páginas admin (Epic 2)

## Dependencies
- Story 1.1 + Story 1.3

## Definition of Done
- [ ] Sidebar renderiza com links corretos
- [ ] Active state funciona por rota
- [ ] `npm run lint` passa

## File List
- `app/(admin)/layout.tsx`
- `components/admin/AdminSidebar.tsx`
