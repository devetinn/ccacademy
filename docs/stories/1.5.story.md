# Story 1.5 — Layout Base Admin + Navegação
**Epic:** EPIC-1 Foundation  
**Status:** Ready for Review  
**Complexity:** S (2 pts)  
**Priority:** P1

## Descrição
Como membro da equipe Carmem, quero um backoffice com navegação clara entre as seções de gerenciamento (trilhas, usuários, Q&A, NPS, métricas).

## Acceptance Criteria

**Given** usuário com role=admin está autenticado  
**When** acessar `/admin/*`  
**Then:**
- [x] Sidebar com itens: Dashboard · Trilhas · Usuários · Perguntas · NPS
- [x] Indicador visual da seção ativa
- [x] Header com "Admin — Carmem Cavalcante Academy" + nome do usuário
- [x] Layout funciona no desktop (min 1024px é o uso principal do admin)
- [x] Breadcrumb em páginas aninhadas (active state indica seção atual)

## Scope
**IN:**
- `app/(admin)/layout.tsx` com sidebar
- Componente AdminSidebar com links e active state

**OUT:**
- Conteúdo das páginas admin (Epic 2)

## Dependencies
- Story 1.1 + Story 1.3

## Definition of Done
- [x] Sidebar renderiza com links corretos
- [x] Active state funciona por rota
- [x] `npm run lint` passa
