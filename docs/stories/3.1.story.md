# Story 3.1 — Home da Academy (Trilhas + Progresso + NPS Banner)
**Epic:** EPIC-3 Core Learning  
**Status:** Ready  
**Complexity:** M (5 pts)  
**Priority:** P0

## Descrição
Como médico cliente, quero ver na home todas as trilhas disponíveis com meu progresso atual, e um banner claro sobre o workshop exclusivo desbloqueável.

## Acceptance Criteria

**Given** médico está autenticado e acessa a home `/`  
**When** a página carregar  
**Then:**
- [ ] Saudação personalizada: "Olá, Dr. [Nome]!"
- [ ] Cards de trilhas com: cover image, título, descrição curta, barra de progresso (%), estimativa de tempo, status (não iniciada/em andamento/concluída)
- [ ] Trilha recomendada em destaque (primeira: "Primeiros Passos no App")
- [ ] Banner NPS Gate: visível para médicos com `nps_answered = false`
  - Título do workshop + descrição + "Responder e desbloquear"
  - Preview de 30s do vídeo embedded
- [ ] Banner NPS Gate: OCULTO para médicos com `nps_answered = true`
- [ ] Loading state com skeleton cards enquanto dados carregam

## Scope
**IN:**
- Listagem de trilhas com progresso real do banco
- Banner NPS Gate (sem o formulário — Story 5.2)
- Skeleton loading

**OUT:**
- Formulário NPS (Story 5.1)
- Lógica de gate do workshop (Story 5.2)

## Dependencies
- Story 1.3 (auth), Story 1.4 (layout), Story 1.2 (dados)

## Definition of Done
- [ ] Progresso real carregado do banco
- [ ] Banner visível para nps_answered=false, oculto para true
- [ ] Responsivo em 375px
- [ ] `npm run typecheck` passa
