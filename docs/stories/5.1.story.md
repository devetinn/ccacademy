# Story 5.1 — Formulário NPS + Trigger de Desbloqueio
**Epic:** EPIC-5 NPS Gate  
**Status:** Ready  
**Complexity:** M (5 pts)  
**Priority:** P0

## Descrição
Como médico cliente, quero responder uma pesquisa rápida (NPS 0–10 + comentário opcional) e desbloquear instantaneamente o Workshop de Reforma Tributária como recompensa.

## Acceptance Criteria

**Given** médico com `nps_answered = false` clica em "Responder e desbloquear"  
**When** interagir com o formulário  
**Then:**
- [ ] Formulário NPS exibido: escala visual 0–10 (botões ou slider), campo de comentário opcional
- [ ] Botão "Enviar e desbloquear" visível
- [ ] Submit: insere em `nps_responses` (score + comment + user_id)
- [ ] Trigger Supabase `after_nps_insert` atualiza `profiles.nps_answered = true`
- [ ] Resposta instantânea: workshop liberado sem reload necessário (optimistic update)
- [ ] E-mail "Workshop desbloqueado" disparado via Resend
- [ ] Formulário some / banner atualizado para "Acesso liberado"

**Given** médico com `nps_answered = true` acessa o workshop  
**When** tentar acessar  
**Then:**
- [ ] Acesso liberado sem formulário

## Scope
**IN:**
- Componente NPSForm (escala 0-10 + comentário)
- Server Action de submissão NPS
- Revalidação imediata do estado nps_answered no client

**OUT:**
- Dashboard de análise do NPS (Story 5.3)

## Dependencies
- Story 1.2 (nps_responses + trigger), Story 4.2 (e-mail)

## Definition of Done
- [ ] Submit → banco atualizado → workshop liberado instantaneamente
- [ ] Trigger Supabase funcionando (nps_answered = true)
- [ ] E-mail disparado
- [ ] `npm run typecheck` passa
