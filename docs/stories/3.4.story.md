# Story 3.4 — Materiais de Apoio + Preview PDF + Download Rastreado
**Epic:** EPIC-3 Core Learning  
**Status:** Ready  
**Complexity:** M (5 pts)  
**Priority:** P0

## Descrição
Como médico cliente, quero baixar e visualizar materiais de apoio de cada aula (PDFs, planilhas, links) sem sair da plataforma. Como admin, quero fazer upload desses materiais pelo CMS.

## Acceptance Criteria

**Given** médico acessa `/aula/[id]` com materiais disponíveis  
**When** visualizar a seção de materiais  
**Then:**
- [ ] Lista de cards de materiais abaixo do player: ícone por tipo (PDF/XLSX/DOCX/link), nome, tamanho
- [ ] Clicar em PDF: abre preview in-browser via react-pdf (sem sair da página)
- [ ] Clicar em XLSX/DOCX: inicia download direto
- [ ] Clicar em link externo: abre em nova aba
- [ ] Download registrado em `attachment_downloads` (user_id + attachment_id + timestamp)
- [ ] Contador de downloads exibido no admin

**Given** admin acessa painel de aula  
**When** gerenciar materiais  
**Then:**
- [ ] Upload de arquivo via Supabase Storage (bucket: `attachments`)
- [ ] Adicionar link externo (URL)
- [ ] Remover material existente

## Scope
**IN:**
- Componente AttachmentList no front da aula
- Preview PDF in-browser (react-pdf)
- Tracking de download em attachment_downloads
- Upload no admin (integrado na Story 2.1 ou como extensão)

**OUT:**
- Editor de PDF
- Conversão de formatos

## Dependencies
- Story 1.2 (tabelas), Story 3.3 (página da aula)

## Definition of Done
- [ ] PDF abre em preview sem sair da página
- [ ] Download registrado no banco
- [ ] Upload funciona no admin
- [ ] `npm run typecheck` passa
