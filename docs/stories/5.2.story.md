# Story 5.2 — Banner NPS Gate na Home + Middleware de Proteção do Workshop
**Epic:** EPIC-5 NPS Gate  
**Status:** Ready  
**Complexity:** M (3 pts)  
**Priority:** P0

## Descrição
Como sistema, preciso exibir o workshop bloqueado de forma atrativa na home e garantir que o middleware bloqueie o acesso ao conteúdo para médicos que não responderam o NPS.

## Acceptance Criteria

**Given** médico com nps_answered=false acessa a home  
**When** visualizar o banner  
**Then:**
- [ ] Banner "Workshop Exclusivo — Desbloqueie Agora" em destaque
- [ ] Exibe: título, descrição, instrutor, duração, preview de 30s (YouTube embed com startTime=0, endTime=30)
- [ ] Botão "Responder pesquisa rápida e desbloquear" → abre modal/drawer com NPSForm

**Given** médico tenta acessar o workshop sem nps_answered=true  
**When** middleware interceptar  
**Then:**
- [ ] Redirecionar para a home com parâmetro `?nps=required`
- [ ] Banner exibido em destaque

**Given** médico com nps_answered=true acessa o workshop  
**When** acessar  
**Then:**
- [ ] Workshop acessível normalmente (aparece como aula normal)

## Scope
**IN:**
- Banner NPSGateBanner atualizado com preview embed
- Middleware estendido para verificar nps_answered em rotas de workshop
- Modal/drawer com NPSForm (Story 5.1)

**OUT:**
- Dashboard NPS (Story 5.3)

## Dependencies
- Story 3.1 (home), Story 5.1 (formulário NPS), Story 1.3 (middleware)

## Definition of Done
- [ ] Preview de 30s funcionando no banner
- [ ] Middleware bloqueando acesso ao workshop sem NPS
- [ ] Fluxo completo: banner → formulário → desbloqueio → acesso funcionando
