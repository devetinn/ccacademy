# Story 5.3 — Dashboard NPS Admin
**Epic:** EPIC-5 NPS Gate  
**Status:** Ready  
**Complexity:** S (2 pts)  
**Priority:** P1

## Descrição
Como membro da equipe Carmem, quero ver o score NPS geral, a distribuição de notas e os comentários qualitativos dos médicos para entender a percepção do serviço.

## Acceptance Criteria

**Given** admin acessa `/admin/nps`  
**When** visualizar o dashboard  
**Then:**
- [ ] Score NPS calculado: ((promotores 9-10) - (detratores 0-6)) / total × 100
- [ ] Distribuição de notas: contagem por score 0–10
- [ ] Lista de comentários com score, data e (nome do médico opcional/anonimizado)
- [ ] Total de respostas e % da base que respondeu

## Scope
**IN:**
- Dashboard simples com queries agregadas
- Sem gráficos animados (tabelas e números são suficientes para V1)

**OUT:**
- Export para CSV
- Filtros por período

## Dependencies
- Story 5.1 (nps_responses no banco), Story 1.5 (layout admin)

## Definition of Done
- [ ] Score NPS calculado corretamente
- [ ] Lista de comentários carregada
