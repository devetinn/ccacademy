# Story 2.1 — CRUD Trilhas + Módulos + Aulas (Admin)
**Epic:** EPIC-2 CMS Admin  
**Status:** Ready for Review  
**Complexity:** L (8 pts)  
**Priority:** P0

## Descrição
Como membro da equipe Carmem, quero criar e gerenciar trilhas, módulos e aulas de forma visual e intuitiva, incluindo reordenação por drag-and-drop.

## Acceptance Criteria

**Given** admin acessa `/admin/trilhas`  
**When** interagir com o CMS  
**Then:**
- [x] Listar todas as trilhas com status (ativa/inativa) e contagem de módulos
- [x] Criar nova trilha: título, descrição, imagem de capa, tempo estimado, status
- [x] Editar trilha existente
- [x] Ativar/desativar trilha (sem deletar)
- [x] Dentro de uma trilha: listar módulos com suas aulas
- [x] Criar módulo: título, associado à trilha
- [x] Dentro de módulo: criar aula: título, descrição, URL do vídeo (YouTube) OU upload, duração, lock_condition
- [x] Reordenar aulas via drag-and-drop (atualiza `order_index` no banco)
- [x] Reordenar módulos via drag-and-drop

## Scope
**IN:**
- CRUD completo de trails, modules, lessons
- Drag-and-drop para reordenação (usar `@hello-pangea/dnd` ou similar)
- Upload de thumbnail para trilha (Supabase Storage)
- Input de URL YouTube para vídeo da aula

**OUT:**
- Upload de vídeo para Supabase Storage (usar YouTube não-listado)
- Gestão de materiais de apoio (Story 2.1b / parte de Story 3.4)

## Dependencies
- Story 1.2 (schema), Story 1.5 (layout admin)

## Definition of Done
- [x] Fluxo completo: criar trilha → módulo → aula funciona
- [x] Drag-and-drop persistindo no banco
- [x] `npm run lint` + `npm run typecheck` passam
